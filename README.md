# color-palette

a color palette algorithm of the site [uicolors](https://uicolors.app/create)

## Usage

### install

```bash
pnpm i @soybeanjs/color-palette
```

### use

```ts
import {
  colorPalettes,
  getColorPalette,
  getColorByColorPaletteNumber,
  getColorName,
} from "@soybeanjs/color-palette";
```
